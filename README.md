# LMSYS Arena Scout

Monitor semanal del estado del arte en Inteligencia Artificial basado en [LMSYS Chatbot Arena](https://lmarena.ai/?leaderboard).

## Estructura

- `index.html` — Dashboard principal (GitHub Pages)
- `data/latest.json` — Datos actuales del ranking
- `scripts/update_data.py` — Script de actualización automática
- `.github/workflows/weekly-update.yml` — Workflow que actualiza datos cada lunes
